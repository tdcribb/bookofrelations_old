To use this sample data, download the ged file to your PC.
Add the tree using the admin Add Tree option.

Set the media paths to:

media path: wp-content/plugins/rootspress/rootspress_data/

portraits path: wp-content/plugins/rootspress/rootspress_data/sample/portraits

Only a few images are included to avoid excessive disk space usage. More can be fount at the support site.
